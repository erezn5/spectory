How to use this program?

This tool is divided to 2 categories:

i. Program tool which acts like Linux shell with the previous commands:
   * mkdir
   * mkfile
   * cd
   * ls
ii. Program test which verify the shell methods and verify its correctness.

while using this tool please pay attention to the following instructions:

1. if you choose to use one of the above actions please notice that for each action there is a unique flow.
   let's take for example 'mkdir' command:
   first you have to choose the command so just type in 'shell>>mkdir'
   secondly type the action itself: 'shell>>mkdir newFolder' --> will lead to a new folder
2. repeat the same for 'mkfile' command and 'cd' command.
3. 'cd' command support '..' operations.
